Location searching:

Right now, returning a valid result with the location search is a bit finicky. You need to type the location as: city name, state, country. The "state" field needs to be a two-letter abbreviation (e.g. Washington -> WA). If the conditions page says "Current conditions for undefined" and no information is loading, a result could not be properly parsed from the location inputed. Make sure to follow the location format shown above. This hopefully be fixed soon.
